## 目录结构

```
src/kernel_eval/
├── __init__.py            # 包入口，导出 Config
├── cli.py                 # CLI 命令入口
├── config.py              # 全局配置
│
├── base/                  # 基类层（抽象定义）
│   ├── __init__.py        # 导出所有基类
│   ├── models.py          # TaskSpec, CaseSpec, InputSpec, OutputSpec, SolutionSpec
│   ├── result.py          # AccuracyResult, PerfResult, OutputResult, compute_speedup
│   ├── loaders.py         # TaskLoader, CaseLoader, GoldenLoaderBase, OperatorDirMixin
│   ├── checker.py         # CorrectnessChecker (抽象基类)
│   ├── matcher.py         # OperatorMatcherBase (抽象基类)
│   ├── scoring.py         # ScoringScheme (抽象基类)
│   └── enums.py           # 枚举定义
│
├── benches/               # CANN 特化层（扁平结构）
│   ├── __init__.py        # 聚合导入，重新导出 CANN 组件
│   ├── cann.py            # 导出 + Registry 注册（核心入口）

│   ├── relative_error_checker.py    # RelativeErrorChecker, RelativeErrorOutputResult

│   ├── __init__.py        # 导出所有 Registry

│   ├── op_runner.py       # OpRunner (算子执行器)

│   ├── results.py         # EvalCaseResult, EvalOperatorResult, EvalSessionResult
```

### 2. Benches 层 (benches/)

CANN 评测集特化实现（扁平结构）：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           benches/ 模块架构                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   cann.py（核心入口）:                                                      │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   1. 导入所有 CANN 特化组件                                          │  │
│   │   2. 注册到 Registry（LoaderRegistry, GoldenLoaderRegistry, ...）    │  │
│   │   3. 执行注册 _register_cann_components()                           │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│   cann_loader.py:                                                           │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   CannTaskLoader   - CANN 任务加载器                                 │  │
│   │   CannCaseLoader   - CANN 用例加载器                                 │  │
│   │   GoldenLoader     - Golden 函数加载器                               │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│   cann_spec.py:                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   CannTaskSpec    - CANN 任务规格（特化字段）                         │  │
│   │   CannCaseSpec    - CANN 用例规格（特化字段）                         │  │
│   │   CannInputSpec   - CANN 输入规格                                    │  │
│   │   CannOutputSpec  - CANN 输出规格                                    │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│   relative_error_checker.py:                                                 │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   RelativeErrorChecker - 相对误差精度判断器（MERE/MARE + CPU fp64）  │  │
│   │   RelativeErrorOutputResult - 相对误差单输出结果                      │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│   cann_matcher.py:                                                          │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   OperatorMatcher - AI 算子匹配器                                    │  │
│   │   (torch.ops.cann_bench 或 cann_bench 模块加载)                     │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│   cann_scoring.py:                                                          │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   CannScoringScheme       - SOL-Score 评分方案                       │  │
│   │   SimpleComparisonScheme  - 加速比评分方案                           │  │

│   │   OperatorScoreInfo       - 算子评分信息                             │  │
│   │   per_case_sol_score      - 单用例 SOL-Score 计算                    │  │

│   └─────────────────────────────────────────────────────────────────────┘  │
```

### 3. Registry 层 (registry/)

注册机制，支持多评测集接入：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           registry/ 模块架构                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   loader_registry.py:                                                       │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   LoaderRegistry                                                    │  │
│   │   - register_task_loader(name, cls)                                 │  │
│   │   - register_case_loader(name, cls)                                 │  │
│   │   - get_task_loader(name, **kwargs)                                 │  │
│   │   - get_case_loader(name, **kwargs)                                 │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│   golden_registry.py:                                                       │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │   GoldenLoaderRegistry                                              │  │
│   │   - register(name, cls)                                             │  │
```
